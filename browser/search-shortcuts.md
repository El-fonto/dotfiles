# Manual setup for site-search shortcuts

Go to:
```
helium://settings/searchEngines
```

Add these:

**Linguee**
- Keyword: ling
- URL: https://www.linguee.com/english-spanish/search?source=auto&query=%s

**AnnasArchive**
- Keyword: annas
- URL: https://annas-archive.li/search?q=%s

**Alternativeto**
- Keyword: altto
- URL:https://alternativeto.net/browse/search/?q=%s

**Google Drive**
- Keyword: dr
- URL: https://drive.google.com/drive/u/0/search?q=%s

**Youtube**
- Keyword: you
- URL: https://www.youtube.com/results?search_query=%s&page={startPage?}&utm_source=opensearch 

**Perplexity**
- Keyword: per
- URL: https://www.perplexity.ai/search/?q=%s

**ESLBrains**
- Keyword: eslb
- URL: https://eslbrains.com/?s=%s
